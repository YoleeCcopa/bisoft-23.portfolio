Mis habilidades

Lenguajes de programación

n (Nivel 2)

  • Capaz de idear funciones/objetos reutilizables que resuelvan el problema general.
  • Capacidad para diseñar sistemas que abarquen múltiples tecnologías/plataformas.
  • El código está agrupado en regiones y bien comentado, con referencias a otros archivos fuente.
  • No hay funciones largas, comentarios que expliquen código inusual, correcciones de errores, suposiciones de código.

n2 (Nivel 1)

  • Capaz de dividir el problema en múltiples funciones.
  • Capaz de dividir el espacio del problema y diseñar la solución siempre que sea dentro de la misma plataforma/tecnología.
  • Los métodos se agrupan lógicamente o por accesibilidad. Buenos nombres para archivos, variables, clases, métodos, etc.

Bases de datos

  • Conoce conceptos básicos de bases de datos, normalización, ACID, transacciones y puede escribir selecciones sencillas.
